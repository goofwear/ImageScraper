# ImageScraper

ImageScraper はディープラーニングに用いる画像を収集する目的で開発された画像ダウンローダです.

## 主な機能

- プラグインによる機能拡張．
- 認証を必要としない多くの Web ページ上の画像のダウンロード．
- サーバー負荷を軽減するためリクエストごとにウェイトを設定．
